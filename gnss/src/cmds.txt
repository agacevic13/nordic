# CLI COMMANDS:

reset - resets the chip. Sample: "reset"
start - starts the ble scanner. Sample: "start"
nwkid - the nwk id that scanner searches for. By default it's 1234 for shell fw. Id length must be 4 at the moment. Sample: "nwkid 2525". 
freq  - Sets the sensor update frequency, in seconds. Sample: "freq 5"
hwid  - Updates hardware id used for applying to a response slot. Most useful for testing many response slots with the same scanner. Sample: "hwid"
